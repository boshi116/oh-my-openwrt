# 使用官方软件包 - 通用即插即用 UPNP

从官方软件仓库中搜素安装

* 软件名: `luci-app-upnp`
* 语言包: `luci-i18n-upnp-zh-cn`

命令行安装

```bash
opkg install luci-app-upnp luci-i18n-upnp-zh-cn
```
