# 使用软件包 - 动态 DDNS

从官方软件仓库中搜素安装

* 软件名: `luci-app-ddns`
* 语言包: `luci-i18n-ddns-zh-cn`

命令行安装

```bash
opkg install luci-app-ddns luci-i18n-ddns-zh-cn
```
