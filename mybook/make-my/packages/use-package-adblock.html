使用软件包 Adblock

Adblock 去广告

安装

opkg update
opkg install luci-i18n-base-zh-cn
opkg install libustream-openssl
opkg install adblock luci-app-adblock luci-i18n-adblock-zh-cn

使用

LuCI ---> 服务 ---> Adblock

勾选启用

勾选 reg_cn 拦截列表，拦截国内广告

LuCI ---> 服务 ---> Adblock ---> 高级 ---> 编辑黑名单
